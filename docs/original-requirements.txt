Step-by-Step Plan

We'll do this over several small phases.

Create a new project:

npx create-next-app@latest ai-project-template

Choose:

✔ TypeScript
✔ App Router
✔ Tailwind CSS
✔ ESLint
✔ src folder → No
✔ Turbopack → Yes
✔ Import Alias → @/*

Run:

npm install @supabase/supabase-js

That means you won't have to paste the same prompt at the start of every conversation.

Your project itself will teach Cursor how to behave.
